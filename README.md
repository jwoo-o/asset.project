# IT자산 웹페이지
## 기획
- 총무 업무를 하며 필요한 데이터들을 엑셀로 공유하는 번거러움에서 시작
- IT자산,사무실 레이아웃,인사관리를 웹페이지로 제작 및 입사일정 공유를 위해 메일 전송 시스템이 필요

## 프로젝트 환경
### 개발 환경
- JDK(8.0)
- Tomcat(7.0) 운영서버(Synology Nas)
- Maria DB 5

### 프로그래밍 언어
- Java(8.0)
- JavaScript
- JSP

### 프레임워크
- Spring(4.0)

### 라이브러리
- Google Chart
- Jquery
- Jquery-ui

### 배포환경
- jenkins
### 형상관리
- github(로컬)
- svn(운영서버 - Synology Nas)
