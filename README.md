# IT자산 웹페이지
## 기획
- 총무 업무를 하며 필요한 데이터들을 엑셀로 공유하는 번거러움에서 시작
- IT자산,사무실 레이아웃,인사관리를 웹페이지로 제작

## 프로젝트 환경

- JDK(8.0)
- NAS Tomcat(7.0)
- Maria DB 5

### 프로그래밍 언어
- Java(8.0)
- JavaScript
- JSP

### 프레임워크
- Spring(4.0)

### 라이브러리

- Jquery
- Jquery-ui

### 배포환경
